
รายละเอียด

ระบบบันทึกรักการอ่าน สำหรับครูและนักเรียน — บันทึก, อนุมัติ, ให้คะแนน และสรุปรายงาน

ฟีเจอร์
- บันทึกการอ่านของนักเรียน
- อนุมัติบันทึกและให้คะแนน
- สรุปรายงานและส่งออกเป็น Excel
- ส่งออกบันทึกเป็นฟอร์มพิมพ์ได้ (HTML/PDF)

การใช้งาน
- เปิดไฟล์ index.html ในเบราว์เซอร์
- **ขั้นแรก**: ตั้งรหัสผ่านครูและ admin ในแอป (ไม่มีค่าตั้งต้นแล้ว)

ฟีเจอร์ใหม่ (อัปเดต)
- **ส่งออก HTML บันทึก**: ครูกดปุ่ม "📄 Export" เพื่อดาวน์โหลดบันทึกในรูปแบบ HTML พร้อมพิมพ์
- **ลบรหัสตัวอย่าง**: ลบครูและ admin ตัวอย่างทั้งหมด

ผู้พัฒนา
