# วิธีใช้ส่งออกบันทึกรักการอ่าน

## ขั้นตอนการส่งออก

### สำหรับครู (ขั้นตอนที่ 1-3)

1. **เข้าสู่ระบบ**
   - คลิก "👩‍🏫 ครูเข้าสู่ระบบ"
   - ใส่รหัสผ่านของครู (เช่น THTUPPT01)

2. **ไปยังหน้า "ตรวจบันทึก"**
   - เมนูด้านซ้าย → "📝 ตรวจบันทึก"
   - จะเห็นรายการบันทึกที่รอการตรวจสอบ

3. **ส่งออกบันทึกแต่ละรายการ**
   - ในตาราง แต่ละแถวจะมีปุ่มสองปุ่ม:
     - **📝 ตรวจ** - เพื่อตรวจและให้คะแนน
     - **📄 Export** - เพื่อส่งออกเป็น HTML/PDF
   - คลิก "📄 Export" เพื่อดาวน์โหลดบันทึก

## การใช้ไฟล์ที่ส่งออก

### การพิมพ์
1. เปิดไฟล์ HTML ในเบราว์เซอร์
2. กด `Ctrl+P` (หรือ `Cmd+P` บน Mac)
3. เลือกตัวเลือกพิมพ์ (ขอบหน้า 0.5 นิ้ว)
4. พิมพ์เป็น PDF หรือ กระดาษโดยตรง

### การบันทึกเป็น PDF
1. เปิดไฟล์ HTML ในเบราว์เซอร์
2. กด `Ctrl+P` (หรือ `Cmd+P` บน Mac)
3. เลือก "Save as PDF" หรือเครื่องพิมพ์เสมือน
4. บันทึกเป็นไฟล์ PDF

## ข้อมูลที่แสดงในบันทึก

บันทึกที่ส่งออกจะแสดงข้อมูลต่อไปนี้:

- **ชื่อหนังสือ** - ชื่อของหนังสือที่นักเรียนอ่าน
- **ผู้แต่ง** - ชื่อผู้แต่งของหนังสือ
- **จำนวนหน้า** - จำนวนหน้าที่อ่าน
- **สำนักพิมพ์** - ชื่อสำนักพิมพ์
- **สรุปเนื้อหา** - ใจความสำคัญของหนังสือ
- **ข้อคิด** - ข้อคิดและบทเรียนที่ได้รับ
- **เหตุผล** - เหตุผลที่เลือกอ่านหนังสือเล่มนี้
- **ข้อมูลนักเรียน** - ชื่อ ห้อง เลขที่ วันที่อ่าน
